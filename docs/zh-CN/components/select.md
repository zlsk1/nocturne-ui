### Select Props

| 属性名                | 描述                   | 类型                                 | 默认值   |
| -------------------- | ---------------------- | ------------------------------------ | ------- |
| modelValue / v-model | 绑定值                  | `string`/ `number`/ `boolean`/ `object`/ `array`| ——       |
| placeholder          | 占位符                  | `string`                             | ——       |
| disabled             | 是否禁用                | `boolean`                             | false    |
