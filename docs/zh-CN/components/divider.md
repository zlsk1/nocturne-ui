## Divider API

### Divider Props

| 属性名      | 描述                    | 类型                                   | 默认值       |
| ----------- | ----------------------- | -------------------------------------- | ------------ |
| dashed      | 是否为虚线              | ^[boolean]                             | false        |
| type        | 分割线类型,垂直还是水平 | ^[enum]`'horizontal' \| 'vertical'`    | 'horizontal' |
| orientation | 分割线标题的位置        | ^[enum]`'left' \| 'center' \| 'right'` | 'center'     |
