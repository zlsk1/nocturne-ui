## Icon API

### Icon Props

| 属性名                | 描述                   | 类型                                 | 默认值   |
| -------------------- | ---------------------- | ------------------------------------ | ------- |
| size                 | 图标尺寸                | ^[string]/ ^[number]                | 24       |
| color                | 颜色                    | ^[string]                            | -       |

### Icon Slots

| 插槽名  | 描述              |
| ------ | ----------------- |
| default| 默认插槽           |
