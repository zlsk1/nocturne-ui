## 安装

选择一个你喜欢的包管理器进行安装

:::code-group

```sh [npm]
$ npm i nocturne-ui --save
```

```sh [pnpm]
$ pnpm i nocturne-ui --save
```

```sh [yarn]
$ yarn add nocturne-ui
```

```sh [bun]
$ bun add nocturne-ui
```

:::

安装失败或者网络异常时可以尝试切换镜像源如 [cnpm](https://github.com/cnpm/cnpm)
